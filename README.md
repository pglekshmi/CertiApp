# CertiApp

## Hardhat Initialisation

```
npx hardhat --init

```

## Hardhat compile

```
npx hardhat compile
```

-- Update .env.eng file as .env and update with your details

## Install dotenv

``` npm i dotenv
```
## Hardhat deploy

```
npx hardhat ignition deploy ./ignition/modules/Cert.ts --network hoodi
```

--Go to React folder

```
npm i
```
Start React

```
npm start
```
